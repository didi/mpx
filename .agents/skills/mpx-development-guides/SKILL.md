# Mpx 开发指南

本 Skill 提供了 Mpx 框架的基础开发指南索引。Mpx 是一款致力于提高小程序开发体验和开发效率的增强型小程序框架。

## 1. Mpx 框架简介
Mpx 是对小程序原生开发标准的补强和扩充，支持单文件开发、数据响应、增强模板语法等。
