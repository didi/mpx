## 样式能力支持情况

> 小程序中以下 CSS 能力在 Web 均可直接使用，无需兼容改造：

- 复合选择器、后代/子代选择器：✅ 支持
- 伪类（`:hover` / `:active` / `:nth-child` / `:first-child` 等）：✅ 支持
- 伪元素（`::before` / `::after` / `::-webkit-scrollbar` 等）：✅ 支持（部分 webkit 私有伪元素小程序不支持，按需隔离）
- 媒体查询 `@media`：✅ 支持
- `grid` / `float` / `position: fixed/sticky`：✅ 支持（注意小程序侧差异，必要时隔离）
- CSS 变量、`calc()`：✅ 支持
